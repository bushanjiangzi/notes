### react 框架的特点

- 声明式开发 - 减少大量 dom 操作
- 可以与其他框架并存
- 组件化
- 单向数据流 - 父组件往子组件传数据，子组件不能改变父组件传递下来的数据
- 视图层框架 - 在大型项目中还需引入 flux，redux 等数据层框架
- 函数式编程 - 易于前端自动化测试

### 生命周期函数

- Initialization
- Mounting
  - componentWillMount - 挂载前
  - render
  - componentDidMount - 挂载完成
- Updation
- UnMounting
