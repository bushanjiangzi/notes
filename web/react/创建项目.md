## 全局安装脚手架
`npm install -g create-react-app`

## 通过脚手架创建项目
- 创建 `npx create-react-app my-app`

- cd `my-app`

- 启动 `npm run start`

- 放开webpack配置 `npm run eject`

- 修改`start`命令语句设置项目端口号: `set PORT=3030 && node scripts/start.js`

- 安装redux: `npm install react-redux redux redux-thunk --save`
