# flow 静态类型检查工具
## 工作方式 ##
- 类型推断
- 类型注释
## 使用
1. 安装

`npm install -g flow-bin`

2. 运行 flow

加@flow说明这个文件要做静态类型检查
```
/* @flow */

function add(x: number, y: number): number {
  if (x, y) {
    return x + y;
  }
}

add(1, 2)
```
