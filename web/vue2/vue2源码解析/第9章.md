## v-model

```
let vm = new Vue({
  el: '#app',
  template: `
    <div>
      <p>Message is {{ message }</p>}
      <input :value="message=$event.target.value" placeholder="edit me">
    </div>
  `,
  data() {
    return {
      message: ''
    }
  }
})
```

- v-model 的本质就是语法糖，但是运行时也做了一些优化

- v-model 既可以支持原生表单元素，也可以支持组件。在组件的实现中，我们是可以配置子组件接收 prop 名称，以及派发的事件名称。
